# AI News Recommendation System using Endee Vector Database

## Overview

This project builds an AI-powered news recommendation system using semantic search.

It converts news articles into embeddings and stores them in the Endee vector database to find similar articles.

## Features

• Semantic search  
• Vector database using Endee  
• AI embeddings using Sentence Transformers  
• Real-time news using NewsData API  
• Interactive UI using Streamlit

## Tech Stack

Python  
SentenceTransformers  
Endee Vector DB  
Streamlit  
NewsData API

## Setup

Install dependencies

pip install -r requirements.txt

Run Endee server

./run.sh

Run application

streamlit run src/app.py
